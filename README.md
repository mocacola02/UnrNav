# UnrNav

Blender 5.0+ plugin designed to bring camera controls similar to UnrealEd 2.0 to Blender. **This extension is currently in alpha, so expect limitations and bugs. Please report any as you find them.**

## How to Use

As of now, UnrNav is its own tool to prevent clashing with other tools that require click & drag. After installing and enabling, select it from the toolbar or press D to enable it. It works in both Object and Edit mode.
If needed, sensitivity can be adjusted from Preferences > Add-ons > UNR Nav.
